## 프로젝트 소개
 - 한강 유역의 수문을 지도 서비스와 연동하여 관측소 위치 및 시설물을 편리하게 볼 수 있습니다.
 - 수자원 시설물 관리시스템을 통해 점검결과 및 이력 조회, 고장/조치 결과 보고서를 작성 할 수 있습니다.<p>
 
### 프로젝트 프로세스
 - 관리자 시스템 : 사용자, 운영, 시스템 관리

 - 수자원 시설물관리 시스템 : 점검이력결과 작성, 조회, 통계 및 고장/조치 보고서<p>
